# LangChain Retrievers

A simple project demonstrating how to use **LangChain Retrievers** for retrieving relevant documents efficiently from a knowledge base. This notebook explains the working of retrievers and how they improve the Retrieval-Augmented Generation (RAG) pipeline.

---

## 📌 Project Overview

This project contains a Jupyter Notebook that demonstrates:

- Introduction to LangChain Retrievers
- Loading documents
- Creating vector embeddings
- Storing embeddings in a vector database
- Retrieving relevant documents
- Understanding similarity search
- Practical implementation of retrieval techniques

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/LangChain_Retrievers.git
```

Move into the project directory:

```bash
cd LangChain_Retrievers
```

Install the required packages:

```bash
pip install langchain
pip install langchain-community
pip install langchain-openai
pip install faiss-cpu
pip install chromadb
pip install python-dotenv
pip install jupyter
```

Or install everything together:

```bash
pip install langchain langchain-community langchain-openai faiss-cpu chromadb python-dotenv jupyter
```
